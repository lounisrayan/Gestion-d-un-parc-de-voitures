Description du projet

Ce projet consiste à créer un programme en Python qui simule la gestion d’un parc de voitures. Le système permet de gérer les voitures présentes dans le parc en respectant des règles précises de capacité et d’organisation.

Il permet d’ajouter et de retirer des voitures selon la capacité maximale du parc et la disponibilité des places.
